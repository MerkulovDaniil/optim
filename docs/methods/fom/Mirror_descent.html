–ú–µ—Ç–æ–¥ –∑–µ—Ä–∫–∞–ª—å–Ω–æ–≥–æ —Å–ø—É—Å–∫–∞ —è–≤–ª—è–µ—Ç—Å—è –µ—Å—Ç–µ—Å—Ç–≤–µ–Ω–Ω—ã–º –æ–±–æ–±—â–µ–Ω–∏–µ–º –º–µ—Ç–æ–¥–∞ –ø—Ä–æ–µ–∫—Ü–∏–∏ —Å—É–±–≥—Ä–∞–¥–∏–µ–Ω—Ç–∞ –≤ —Å–ª—É—á–∞–µ –æ–±–æ–±—â–µ–Ω–∏—è l_2 –Ω–æ—Ä–º—ã –Ω–∞ –±–æ–ª–µ–µ –æ–±—â–∏–π —Å–ª—É—á–∞–π –∫–∞–∫–æ–π-—Ç–æ —Ñ—É–Ω–∫—Ü–∏–∏ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è.

0.1 Dual norm:

–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ: –°–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–π –Ω–æ—Ä–º–æ–π \|\cdot\|_* –∫ –¥–∞–Ω–Ω–æ–π \|\cdot\| –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è:

\|y\|_* = \max \{\langle y,x\rangle: \|x\|=1 \}

–ü—Ä–∏–º–µ—Ä: (\|\cdot \|_p)_* = \|\cdot \|_q, \qquad \dfrac{1}{p} + \dfrac{1}{q} = 1

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:

–ù–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –ì–µ–ª—å–¥–µ—Ä–∞:

\sum_{k=1}^n |x_k\,y_k| \le \biggl( \sum_{k=1}^n |x_k|^p \biggr)^{\frac{1}{p}} \biggl( \sum_{k=1}^n |y_k|^q \biggr)^{\frac{1}{q}} \text{ for all }x, y \in \mathbb{C}^n

–°–≤–æ–π—Å—Ç–≤–∞:

  • –î–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–∞—è –Ω–æ—Ä–º–∞ \|\cdot\|_* —è–≤–ª—è–µ—Ç—Å—è –Ω–æ—Ä–º–æ–π
  • l_2 –Ω–æ—Ä–º–∞ —Å–æ–ø—Ä—è–∂–µ–Ω–∞ —Å–∞–º–∞ —Å–µ–±–µ
  • –î–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–∞—è –Ω–æ—Ä–º–∞ –∫ –¥–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–æ–π –Ω–æ—Ä–º–µ - –∏—Å—Ö–æ–¥–Ω–∞—è –Ω–æ—Ä–º–∞
  • (\|\cdot\|_1)_* = \|\cdot\|_\infty, \;(\|\cdot\|_\infty)_* = \|\cdot\|_1
  • –û–±–æ–±—â–µ–Ω–Ω–æ–µ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –ö–æ—à–∏ –®–≤–∞—Ä—Ü–∞: \langle y,x \rangle \leq \|y\|_*\|x\|, —Å–ª–µ–¥—Å—Ç–≤–∏–µ: \|x\|^2 \pm 2 \langle y,x \rangle + \|y\|_*^2 \geq 0

0.2 Bregman divergence

–ü–æ–ø—Ä–æ–±—É–µ–º –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω–æ –≤–≤–µ—Å—Ç–∏ –ø–æ–Ω—è—Ç–∏–µ –æ–±–æ–±—â–µ–Ω–Ω–æ–≥–æ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è, –∏–º–µ–Ω—É–µ–º–æ–≥–æ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ–º –ë—Ä—ç–≥–º–∞–Ω–∞. –î–ª—è –∫–∞–∂–¥–æ–π —Ç–æ—á–∫–∏ y –æ–Ω–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ —ç—Ç–æ–π —Ç–æ—á–∫–∏ –¥–æ x - V_x(y). –í —Å–∞–º–æ–º –ø—Ä–æ—Å—Ç–æ–º —Å–ª—É—á–∞–µ –º–æ–∂–Ω–æ –≤–∑—è—Ç—å V_x(y) = \frac{1}{2}\|x-y\|^2, \;\; \nabla V_x(y) = y-x. –†–∞—Å—Å–º–æ—Ç—Ä–∏–º —É–∂–µ –∫–ª–∞—Å—Å–∏—á–µ—Å–∫—É—é –∑–∞–ø–∏—Å—å:

\begin{align*} \|x_{k+1} - y\|^2 &= \|x_{k+1}-x_k \|^2 + \|x_k - y\|^2 - 2 \langle x_k - x_{k+1} ,x_k - y\rangle \\ \tag{Req1} V_{x_{k+1}}(y) &= V_{x_{k+1}}(x_k) + V_{x_{k}}(y) - \langle \nabla V_{x_{k+1}(x_k)}, x_k - y \rangle \end{align*}

–î–ª—è –≤–≤–æ–¥–∏–º–æ–≥–æ –æ–±–æ–±—â–µ–Ω–Ω–æ–≥–æ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è –±—É–¥–µ–º —Ç—Ä–µ–±–æ–≤–∞—Ç—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è (Req1), –∫—Ä–æ–º–µ —Ç–æ–≥–æ (–∫–∞–∫ –±—É–¥–µ—Ç –≤–∏–¥–Ω–æ –ø—Ä–∏ –ø–æ–ª—É—á–µ–Ω–∏–∏ –æ—Ü–µ–Ω–æ–∫), –ø—Ä–∏—è—Ç–Ω—ã–º —Å–≤–æ–π—Å—Ç–≤–æ–º –±—ã–ª–æ –±—ã –µ—â–µ —Å–ª–µ–¥—É—é—â–µ–µ —Ç—Ä–µ–±–æ–≤–∞–Ω–∏–µ:

\tag{Req2} V_x(y) \geq \frac{1}{2} \|x-y\|^2

–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ: –î–∏–≤–µ—Ä–≥–µ–Ω—Ü–∏–µ–π (—Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ–º) –ë—Ä—ç–≥–º–∞–Ω–∞ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Ñ—É–Ω–∫—Ü–∏—è —Å–ª–µ–¥—É—é—â–∞—è V_x(y). –ü—É—Å—Ç—å S \subseteq \mathbb{R}^n - –∑–∞–º–∫–Ω—É—Ç–æ–µ –≤—ã–ø—É–∫–ª–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —Ç–æ–≥–¥–∞ —Ñ—É–Ω–∫—Ü–∏—è \phi : S \to \mathbb{R} –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –ø—Ä–æ–∫—Å-—Ñ—É–Ω–∫—Ü–∏–µ–π (distance generating function), –µ—Å–ª–∏ \phi —è–≤–ª—è–µ—Ç—Å—è 1 - —Å–∏–ª—å–Ω–æ –≤—ã–ø—É–∫–ª–æ–π, —Ç.–µ.:

\phi(y) \geq \phi(x) + \langle \nabla \phi(x), y-x\rangle + \frac{1}{2} \|y-x\|^2, \qquad \forall x,y \in S

–¢–æ–≥–¥–∞ –ø—Ä–æ–∫—Å-—Ñ—É–Ω–∫—Ü–∏–µ–π –∏–Ω–¥—É—Ü–∏—Ä—É–µ—Ç—Å—è —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –ë—Ä—ç–≥–º–∞–Ω–∞:

V_x(y) = \phi(y) - \phi(x) - \langle\nabla \phi(x), y-x\rangle

–ó–∞–º–µ—Ç–∏–º, —á—Ç–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Å–∏–ª—å–Ω–æ–π –≤—ã–ø—É–∫–ª–æ—Å—Ç–∏ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç –≤—ã–±–æ—Ä–∞ –ø—Ä—è–º–æ–π –Ω–æ—Ä–º—ã \|\cdot\|. –≠—Ç–æ –≤–∞–∂–Ω–æ–µ –∑–∞–º–µ—á–∞–Ω–∏–µ, –ø–æ—Å–∫–æ–ª—å–∫—É –∏–º–µ–Ω–Ω–æ —ç—Ç–æ —Å–≤–æ–π—Å—Ç–≤–æ –ø–æ–∑–≤–æ–ª–∏—Ç –≤ –±—É–¥—É—â–µ–º –ø–æ–¥—Å—Ç—Ä–∞–∏–≤–∞—Ç—å —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –ø–æ–¥ –≥–µ–æ–º–µ—Ç—Ä–∏—é –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞.

0.2.1 Examples

  • –í—ã–±–µ—Ä–µ–º –Ω–æ—Ä–º—É –≤ –ø—Ä—è–º–æ–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ \|\cdot\| = \|\cdot\|_2, –ø—É—Å—Ç—å \phi(x) = \frac{1}{2}\|x\|^2, —Ç–æ–≥–¥–∞ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –ë—Ä—ç–≥–º–∞–Ω–∞ V_x(y) = \frac{1}{2}\|x-y\|^2. –¢–∞–∫–æ–π –≤—ã–±–æ—Ä —Å–æ–≤–ø–∞–¥–∞–µ—Ç —Å —Ç–µ–º, —á—Ç–æ –º—ã –≤–∏–¥–µ–ª–∏ —Ä–∞–Ω–µ–µ –≤ –º–µ—Ç–æ–¥–µ –ø—Ä–æ–µ–∫—Ü–∏–∏ —Å—É–±–≥—Ä–∞–¥–∏–µ–Ω—Ç–∞
  • –í—ã–±–µ—Ä–µ–º —Ç–µ–ø–µ—Ä—å –¥—Ä—É–≥—É—é –Ω–æ—Ä–º—É \|\cdot\| = \| \cdot \|_1, –ø—É—Å—Ç—å \phi(x) = \sum\limits_{i \in [n]}x_i \log x_i - –∞–Ω—Ç–∏—ç–Ω—Ç—Ä–æ–ø–∏—è. –¢–æ–≥–¥–∞ —ç—Ç–∞ —Ñ—É–Ω–∫—Ü–∏—è –±—É–¥–µ—Ç 1 —Å–∏–ª—å–Ω–æ –≤—ã–ø—É–∫–ª–∞ –Ω–∞ –≤—ã–ø—É–∫–ª–æ–º –º–Ω–æ–∂–µ—Å—Ç–≤–µ S : \left\{x \in S : x \geq 0, \sum\limits_{i \in [n]} x_i = 1\right\} (–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–Ω–æ–º —Å–∏–º–ø–ª–µ–∫—Å–µ), –∞ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∞—è –µ–π –¥–∏–≤–µ—Ä–≥–µ–Ω—Ü–∏—è –ë—Ä—ç–≥–º–∞–Ω–∞: V_x(y) = \sum\limits_{i \in [n]} y_i \log \frac{y_i}{x_i} = D(y \| x) - —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –ö—É–ª—å–±–∞–∫–∞ - –õ—è–π–±–ª–µ—Ä–∞.
  • –ï—â–µ –Ω–µ–º–Ω–æ–≥–æ –ø—Ä–∏–º–µ—Ä–æ–≤ –æ—Ç—Å—é–¥–∞:

0.2.2 –°–≤–æ–π—Å—Ç–≤–∞

  • –ê–∫—Å–∏–æ–º–∞ —Ç–æ–∂–¥–µ—Å—Ç–≤–∞ V_x(x) = 0
  • –°–æ–≤–º–µ—Å—Ç–∏–º–æ—Å—Ç—å —Å –ï–≤–∫–ª–∏–¥–æ–≤–æ–π –Ω–æ—Ä–º–æ–π: V_x(y) \geq \frac{1}{2}\|x-y\|^2 \geq 0
  • (–ù–µ)—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ —Ç—Ä–µ—É–≥–æ–ª—å–Ω–∏–∫–∞: \langle -\nabla V_x(y), y-z\rangle = V_x(z) - V_y(z) - V_x(y)

–ü–µ—Ä–≤—ã–µ –¥–≤–∞ —Å–≤–æ–π—Å—Ç–≤–∞ –æ—á–µ–≤–∏–¥–Ω—ã –∏–∑ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è. –î–æ–∫–∞–∂–µ–º —Ç—Ä–µ—Ç—å–µ:

\begin{align*} \langle -\nabla V_x(y), y-z\rangle &= \langle \nabla \phi(x) - \nabla \phi(y) , y-z\rangle =\\ & = (\phi(z) - \phi(x) - \langle \nabla \phi(x), z -x \rangle) \\ & - (\phi(z) - \phi(y) - \langle \nabla \phi(y), z - y \rangle) \\ & - (\phi(y) - \phi(x) - \langle \nabla \phi(x), y - x \rangle) \\ & = V_x(z) - V_y(z) - V_x(y) \end{align*}

1 –í–æ–∑–≤—Ä–∞—â–µ–Ω–∏–µ –∫ –∏—Å—Ç–æ–∫–∞–º

–ü—É—Å—Ç—å –∑–∞–¥–∞–Ω–æ –≤—ã–ø—É–∫–ª–æ–µ –∑–∞–º–∫–Ω—É—Ç–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ S \in \mathbb{R}^n, –∫—Ä–æ–º–µ —Ç–æ–≥–æ, –µ—Å—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏, –≤–æ–∑–≤—Ä–∞—â–∞—é—â–∏–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ç–æ—á–µ–∫ x_1, \ldots, x_k, \ldots. –¢–æ–≥–¥–∞ –∑–∞–ø–∏—à–µ–º (–Ω–µ)—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ —Ç—Ä–µ—É–≥–æ–ª—å–Ω–∏–∫–∞ –¥–ª—è —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è –ë—Ä—ç–≥–º–∞–Ω–∞, –ø–æ–ª–∞–≥–∞—è y = x_{k+1}, x = x_k –∏ –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã–π z \in S (–∫–æ—Ç–æ—Ä—ã–π –º—ã –≤ –¥–∞–ª—å–Ω–µ–π—à–µ–º –¥–ª—è —Ü–µ–ª–æ—Å—Ç–Ω–æ—Å—Ç–∏ –∏–∑–ª–æ–∂–µ–Ω–∏—è –±—É–¥–µ–º –æ–±–æ–∑–Ω–∞—á–∞—Ç—å y)

\begin{align*} \langle -\nabla V_{x_k}(x_{k+1}), x_{k+1}-z\rangle &= V_{x_k}(z) - V_{x_{k+1}}(z) - V_{x_k}(x_{k+1}) \\ \tag{baseMD} \langle -\nabla V_{x_k}(x_{k+1}), x_{k+1}-y\rangle &= V_{x_k}(y) - V_{x_{k+1}}(y) - V_{x_k}(x_{k+1}) \end{align*}

–ü—Ä–æ—Å—É–º–º–∏—Ä—É–µ–º –ø–æ–ª—É—á–µ–Ω–Ω—ã–µ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞:

\sum\limits_{k = 0}^{T-1}\langle -\nabla V_{x_k}(x_{k+1}), x_{k+1}-y\rangle = V_{x_0}(y) - V_{x_{T}}(y) - \sum\limits_{k = 0}^{T-1}V_{x_k}(x_{k+1})

–ò–º–µ—è –≤–≤–∏–¥—É –ø–æ–ª—É—á–µ–Ω–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ, –¥–∞–≤–∞–π—Ç–µ, –Ω–∞–∫–æ–Ω–µ—Ü, –ø–æ–ø—Ä–æ–±—É–µ–º —Å—Ñ–æ—Ä–º—É–ª–∏—Ä–æ–≤–∞—Ç—å –º–µ—Ç–æ–¥ –∑–µ—Ä–∫–∞–ª—å–Ω–æ–≥–æ —Å–ø—É—Å–∫–∞:

x_{k+1} = \text{arg}\min\limits_{x \in S} \left( \langle \alpha_k g_k, x \rangle + V_{x_k}(x) \right)

–ü–æ—Å–º–æ—Ç—Ä–∏–º –≤–Ω–∏–º–∞—Ç–µ–ª—å–Ω–µ–µ –Ω–∞ —É—Å–ª–æ–≤–∏–µ –ø—Ä–æ–µ–∫—Ü–∏–∏ –¥–ª—è —Ç–æ—á–∫–∏ x_{k+1}:

\langle \alpha_k g_k,x_{k+1} - y\rangle + \langle \nabla V_{x_k}(x_{k+1}),x_{k+1} - y\rangle \leq 0

\langle \alpha_k g_k,x_{k+1} - y\rangle \leq - \langle \nabla V_{x_k}(x_{k+1}),x_{k+1} - y\rangle

–ü–æ–ø—Ä–æ–±—É–µ–º —Ç–µ–ø–µ—Ä—å –ø–æ–ª—É—á–∏—Ç—å –Ω–∞—à–µ –±–∞–∑–æ–≤–æ–µ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ, –∏—Å–ø–æ–ª—å–∑—É—è (baseMD):

\begin{align*} \langle \alpha_k g_k, x_k - y\rangle &\leq - \langle \nabla V_{x_k}(x_{k+1}),x_{k+1} - y\rangle - \langle \alpha_k g_k, x_{k+1} - x_k\rangle = \\ & = V_{x_k}(y) - V_{x_{k+1}}(y) - V_{x_k}(x_{k+1})- \langle \alpha_k g_k, x_{k+1} - x_k\rangle \leq\\ &\leq V_{x_k}(y) - V_{x_{k+1}}(y) - \frac{1}{2}\|x_k - x_{k+1}\|^2- \langle \alpha_k g_k, x_{k+1} - x_k\rangle \leq \\ &\leq V_{x_k}(y) - V_{x_{k+1}}(y) + \left(\langle \alpha_k g_k, x_k - x_{k+1}\rangle- \frac{1}{2}\|x_k - x_{k+1}\|^2 \right) \leq \\ &\leq V_{x_k}(y) - V_{x_{k+1}}(y) + \frac{\alpha_k^2}{2} \|g_k\|_*^2 \end{align*}

–¢–ï–õ–ï–°–ö–û–ü–ò–†–£–ï–ú

\begin{align*} \sum\limits_{k = 0}^{T-1} \langle \alpha_k g_k, x_k - y\rangle &\leq V_{x_0}(y) - V_{x_{T}}(y) + \sum\limits_{k = 0}^{T-1}\frac{\alpha_k^2}{2} \|g_k\|_*^2 \\ &\leq V_{x_0}(y) + \sum\limits_{k = 0}^{T-1}\frac{\alpha_k^2}{2} \|g_k\|_*^2 \\ &\leq M + \dfrac{\alpha^2 G^2 T}{2} \end{align*}

–ó–¥–µ—Å—å –º—ã –ø–æ–¥—Ä–∞–∑—É–º–µ–≤–∞–µ–º \|g_k\|_* \leq G —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ –ø–æ k, –∞ V_{x_0}(y) \leq M. –í –∏—Ç–æ–≥–µ:

\begin{align*} f(\overline{x}) - f^* &= f \left( \frac{1}{T}\sum\limits_{k=0}^{T-1} x_k \right) - f^* \leq \dfrac{1}{T} \left( \sum\limits_{k=0}^{T-1} f(x_k) - f^* \right) \\ & \leq \dfrac{1}{T} \left( \sum\limits_{k=0}^{T-1}\langle g_k, x_k - x^* \rangle\right) \\ & \leq \dfrac{M}{\alpha T} + \dfrac{\alpha G^2}{2} \leq \sqrt{\dfrac{2 M G^2}{T}} \end{align*}

–í—ã–±–∏—Ä–∞—è —à–∞–≥ \alpha_k = \alpha = \sqrt{\dfrac{2M}{G^2 T}}

1.1 –ê–ª–≥–æ—Ä–∏—Ç–º –∑–µ—Ä–∫–∞–ª—å–Ω–æ–≥–æ —Å–ø—É—Å–∫–∞ (mirror descent):

x_{k+1} = \text{arg}\min\limits_{x \in S} \left( \langle \alpha_k g_k, x \rangle + V_{x_k}(x) \right)

–ò–Ω—Ç–µ—Ä–µ—Å–Ω—ã–µ —Ñ–∏—à–∫–∏:

1.2 –û–Ω–ª–∞–π–Ω –≤–µ—Ä—Å–∏—è

–°–æ–≤–µ—Ä—à–µ–Ω–Ω–æ —è—Å–Ω–æ, —á—Ç–æ –≤ –Ω–∞—à–∏—Ö –æ—Ü–µ–Ω–∫–∞—Ö –Ω–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–æ–≤–∞—è —Ñ—É–Ω–∫—Ü–∏—è f_k(x) –Ω–∞ –∑–∞–¥–∞–Ω–Ω–æ–º –∫–ª–∞—Å—Å–µ. –ü–æ—ç—Ç–æ–º—É, –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã–µ –æ—Ü–µ–Ω–∫–∏ –ø–æ–ª—É—á–∞—é—Ç—Å—è –∏ –¥–ª—è –æ–Ω–ª–∞–π–Ω –ø–æ—Å—Ç–∞–Ω–æ–≤–∫–∏:

R_{T-1} = \sum\limits_{k = 0}^{T-1} f_k(x_k) - \min_{x} \sum\limits_{k = 0}^{T-1} f_k(x) \leq \sqrt{2 M G^2 T}

1.3 –ï—â–µ –æ–¥–Ω–∞ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è

–î–∞–≤–∞–π—Ç–µ –ø–æ–∫–∞–∂–µ–º, —á—Ç–æ –ø–æ–ª—É—á–µ–Ω–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º –∏–º–µ–µ—Ç –µ—â–µ –æ–¥–Ω—É –æ—á–µ–Ω—å –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω—É—é –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—é:

  1. y_{k} = \nabla \phi(x_k) –û—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ –≤ —Å–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ \nabla \phi(x)
  2. y_{k+1} = y_k - \alpha_k \nabla f_k(x_k) –ì—Ä–∞–¥–∏–µ–Ω—Ç–Ω—ã–π —à–∞–≥ –≤ —Å–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ
  3. x_{k+1} = \text{arg}\min\limits_{x \in S}V_{\nabla \phi^*(y_{k+1})}(x) –û–±—Ä–∞—Ç–Ω–æ–µ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ \nabla \phi^*(x) –∏ –ø—Ä–æ–µ–∫—Ü–∏—è –Ω–∞ –±—é–¥–∂–µ—Ç–Ω–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ

–î–ª—è –¥–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–∞ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ—Å—Ç–∏ —Ç–∞–∫–∏—Ö –∑–∞–ø–∏—Å–µ–π, —Å–ª–µ–¥—É–µ—Ç —Å–Ω–∞—á–∞–ª–∞ –¥–æ–∫–∞–∑–∞—Ç—å —Ñ–∞–∫—Ç —Ç–æ–≥–æ, —á—Ç–æ:

\left( \nabla \phi(x) \right)^{-1} = \nabla \phi^*(y)

–î–ª—è —ç—Ç–æ–≥–æ –ø—É—Å—Ç—å y = \nabla \phi(x). –ó–∞–º–µ—Ç–∏–º, —á—Ç–æ –¥–ª—è —Å–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤–æ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –§–µ–Ω—Ö–µ–ª—è - –Æ–Ω–≥–∞: \phi^*(y) + \phi(x) \geq xy, –≤ —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ \phi(x) - –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä—É–µ–º–∞, —Ç–∞–∫–æ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ–º –õ–µ–∂–∞–Ω–¥—Ä–∞ –∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ: \phi^*(y) + \phi(x) = xy. –î–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä—É—è —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –ø–æ y, –ø–æ–ª—É—á–∞–µ–º \nabla \phi^*(y) = x. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º,

\nabla\phi^*(y) = \nabla\phi^*(\nabla \phi(x)) = x, \qquad \nabla\phi(x) = \nabla\phi(\nabla \phi^*(y)) = y

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:

\begin{align*} x_{k+1} &= \text{arg}\min\limits_{x \in S} \left\{ V_{\nabla \phi^*(y_{k+1})}(x) \right\} = \\ &= \text{arg}\min\limits_{x \in S} \left\{ \phi(x) - \phi(\nabla \phi^*(y_{k+1})) - \left\langle \nabla \phi (\nabla \phi^*(y_{k+1})),x - \nabla \phi^*(y_{k+1})\right\rangle\right\} = \\ &= \text{arg}\min\limits_{x \in S} \left\{ \phi(x) - \left\langle y_{k+1},x \right\rangle \right\} = \\ &= \text{arg}\min\limits_{x \in S} \left\{ \phi(x) - \left\langle \nabla \phi(x_k) - \alpha_k g_k,x \right\rangle \right\} = \\ &= \text{arg}\min\limits_{x \in S} \left\{ \phi(x) - \phi(x_k) - \left\langle \nabla \phi(x_k),x \right\rangle + \left\langle \alpha_k g_k,x \right\rangle \right\} = \\ &= \text{arg}\min\limits_{x \in S} \left\{ V_{x_k}(x) + \left\langle \alpha_k g_k,x \right\rangle \right\} \end{align*}

–í –ø–æ—Å–ª–µ–¥–Ω–µ–π —Å—Ç—Ä–æ—á–∫–µ –º—ã –ø—Ä–∏—à–ª–∏ –∫ —Ç–æ–π —Ñ–æ—Ä–º—É–ª–∏—Ä–æ–≤–∫–µ, –∫–æ—Ç–æ—Ä—É—é –ø–∏—Å–∞–ª–∏ —Ä–∞–Ω—å—à–µ. –ó–∞–º–µ—Ç–∏–º —Ç–∞–∫ –∂–µ, –µ—â–µ –æ–¥–Ω—É –∏–Ω—Ç–µ—Ä–µ—Å–Ω—É—é –∫–æ–Ω—Ü–µ–ø—Ü–∏—é:

\begin{align*} x_{k+1} &= \text{arg}\min\limits_{x \in S} \left( \langle \alpha_k g_k, x \rangle + V_{x_k}(x) \right) \\ &= \text{arg}\min\limits_{x \in S} \left( \langle g_k, x \rangle + \frac{1}{\alpha_k}V_{x_k}(x) \right) \\ &= \text{arg}\min\limits_{x \in S} \left(f(x_k)+ \langle g_k, x \rangle + \frac{1}{\alpha_k}V_{x_k}(x) \right) \end{align*}

–ó–¥–µ—Å—å –ª–µ–≤–∞—è —á–∞—Å—Ç—å –º–∏–Ω–∏–º–∏–∑–∏—Ä—É–µ–º–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –∞–ø–ø—Ä–æ–∫—Å–∏–º–∞—Ü–∏—é –ø–µ—Ä–≤–æ–≥–æ –ø–æ—Ä—è–¥–∫–∞, –∞ –ø—Ä–∞–≤–∞—è —á–∞—Å—Ç—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –ø—Ä–æ–µ–∫—Ü–∏–æ–Ω–Ω—ã–π —á–ª–µ–Ω.

2 –ù–ê–§–ò–ì–ê?

–†–µ–∑–æ–Ω–Ω—ã–π –≤–æ–ø—Ä–æ—Å, –≤–µ–¥—å –≤ —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ –º—ã –≤—ã–±—Ä–∞–ª–∏ \|\cdot\| = \|\cdot\|_2 –ï–≤–∫–ª–∏–¥–æ–≤—É –Ω–æ—Ä–º—É –∏ –ï–≤–∫–ª–∏–¥–æ–≤–æ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ, —Ç–æ —ç—Ç–æ—Ç –º–µ—Ç–æ–¥ –≤ —Ç–æ—á–Ω–æ—Å—Ç–∏ —Å–æ–≤–ø–∞–¥–∞–µ—Ç —Å —Ç–µ–º, —á—Ç–æ –º—ã —É–∂–µ –Ω–∞–∑—ã–≤–∞–µ–º –º–µ—Ç–æ–¥ –ø—Ä–æ–µ–∫—Ü–∏–∏ —Å—É–±–≥—Ä–∞–¥–∏–µ–Ω—Ç–∞.

–ó–Ω–∞—á–∏—Ç, –Ω–∞–¥–æ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–∏—Ç—å —Å—Ü–µ–Ω–∞—Ä–∏–π, –∫–æ–≥–¥–∞ –ú–ó–° —Ä–∞–±–æ—Ç–∞–µ—Ç –ª—É—á—à–µ, –¥–∞–≤–∞–π—Ç–µ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º S = \Delta_n - –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–Ω—ã–π —Å–∏–º–ø–ª–µ–∫—Å, –∞ —Ç–∞–∫ –∂–µ —Å–ª–µ–¥—É—é—â–µ–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –ë—Ä—ç–≥–º–∞–Ω–∞ V_x(y) = \sum_{i \in [n]} y_i \log \frac{y_i}{x_i} = D(y \| x). –ù–æ—Ä–º–∞ –≤ –ø—Ä—è–º–æ–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ –ø—Ä–∏ —ç—Ç–æ–º \|\|_1, –∞ –≤ —Å–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–º - \|\|_\infty. –ö—Ä–æ–º–µ —Ç–æ–≥–æ, –¥–ª—è –∑–∞–¥–∞–Ω–Ω–æ–π –¥–∏–≤–µ—Ä–≥–µ–Ω—Ü–∏–∏ –ë—Ä—ç–≥–º–∞–Ω–∞ —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤–æ:

x_0 = \left( 1/n, \ldots, 1/n \right) \; \to \; V_{x_0}(x) \leq \log n \;\;\forall x \in \Delta_n

–¢–æ–≥–¥–∞ –∞–ª–≥–æ—Ä–∏—Ç–º –∑–µ—Ä–∫–∞–ª—å–Ω–æ–≥–æ —Å–ø—É—Å–∫–∞ –∑–∞–ø–∏—à–µ—Ç—Å—è –≤ –≤–∏–¥–µ:

\begin{align*} x_{k+1} &= \text{arg}\min\limits_{x \in S} \left( \langle \alpha_k g_k, x \rangle + V_{x_k}(x) \right) \\ &= \text{arg}\min\limits_{x \in S} \left( \langle \alpha_k g_k, x \rangle + \sum_{i \in [n]} x_i \log \frac{x_i}{x_{k_i}} \right) \\ &= x_k \cdot \dfrac{e^{-\alpha_k g_k}}{\|x_k \cdot e^{-\alpha_k g_k}\|_1} \end{align*}

–ê –æ—Ü–µ–Ω–∫–∏ —Å —É—á–µ—Ç–æ–º —Ç–æ–≥–æ, —á—Ç–æ M = \log n, \|g_k\|_\infty \leq G –∑–∞–ø–∏—à—É—Ç—Å—è, –∫–∞–∫:

\begin{align*} f(\overline{x}) - f^* \leq \sqrt{\dfrac{2 \log (n) G^2}{T}} \end{align*} 

import numpy as np
from matplotlib import pyplot as plt
%matplotlib inline
import seaborn as sns
sns.set()


Ts = np.logspace(0,4, 10)

m = 10
n = 1000
A = np.random.randn(m, n)
x_true = np.random.randn(n)
x_true[x_true < 0] = 0
x_true = x_true/(np.linalg.norm(x_true, 1))

b = A@x_true

x0 = np.ones(n)/n


def f(x):
    return np.linalg.norm(A@x - b, 1)

def grad(x):
    return np.sum(A.T * np.sign(A@x - b), axis=1)

def mirror_descent(x0, grad, T):
    n = len(x0)
    M = np.log(n)
##     G = np.linalg.norm(A,np.inf)*1+np.linalg.norm(b,np.inf)
##     alpha = np.sqrt(2*M/(G**2*T))
    alpha = 0.0001
    xk = x0
    sequence = []
##     print('MD %.3f'%alpha)
    for i in range(int(T)):
        sequence.append(xk)
        g = grad(xk)
        xk = xk * np.exp(-alpha * g) / np.sum(xk * np.exp(-alpha * g))
    return sequence

def projection_subgradient_descent(x0, grad, T):
    n = len(x0)
    M = 0.5
##     G = np.linalg.norm(A,2)*1+np.linalg.norm(b,2)
##     alpha = np.sqrt(2*M/(G**2*T))
    alpha = 0.0001
##     print('GD %.3f'%alpha)
    xk = x0
    sequence = []
    for i in range(int(T)):
        sequence.append(xk)
        g = grad(xk) 
        xk = xk - alpha*g
        xk[xk<0] = 0
        xk = xk/(np.linalg.norm(xk, 1))
    return sequence

result_md = []
result_gd = []

for T in Ts:
    print(T)
    md_T = mirror_descent(x0, grad, T)
    gd_T = projection_subgradient_descent(x0, grad, T)
    
    x_md = np.mean(md_T, axis = 0)
    x_gd = np.mean(gd_T, axis = 0)
    
    result_md.append(f(x_md) - f(x_true))
    result_gd.append(f(x_gd) - f(x_true))

    
plt.loglog(Ts, result_gd, label = 'GD')
plt.loglog(Ts, result_md, label = 'MD')
